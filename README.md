**Ghid complet Claude Code in limba romana — 10 module, de la incepator la avansat.**

Claude Code este agentul de programare creat de Anthropic care traieste in terminalul tau. Acest ghid te invata sa il folosesti de la zero pana la nivel avansat, cu explicatii practice, configurari copy-paste si exercitii pe care le poti aplica imediat.

Nu este o traducere — este un ghid original, scris din perspectiva unui utilizator activ, pentru comunitatea romaneasca de developeri.

## Auto-evaluare — De unde incepi?

Raspunde sincer la aceste intrebari. Numara cate raspunsuri afirmative ai:

1. Stii sa pornesti Claude Code si sa ii dai o instructiune in limbaj natural?
2. Ai configurat un fisier `CLAUDE.md` intr-un proiect?
3. Folosesti `/compact`, `/diff` sau `/clear` in mod regulat?
4. Ai creat cel putin o comanda slash personalizata?
5. Stii ce sunt hooks si ai configurat cel putin unul?
6. Ai conectat Claude Code la un serviciu extern prin MCP?
7. Ai folosit sub-agenti sau planning mode?
8. Ai creat sau instalat un plugin?

**Rezultat:**
- **0-2 raspunsuri:** Incepe cu [Nivel Incepator](#nivel-incepator) (module 01-04)
- **3-5 raspunsuri:** Treci direct la [Nivel Intermediar](#nivel-intermediar) (module 05-08)
- **6-8 raspunsuri:** Du-te la [Nivel Avansat](#nivel-avansat) (module 09-10)

## Parcursuri de invatare

### Nivel Incepator
**Durata:** ~2.5 ore | **Module:** 01, 02, 03, 04

Inveti fundamentele: cum comunici cu Claude Code, cum ii dai context permanent prin CLAUDE.md, cum te protejezi cu checkpoints si ce comenzi ai la dispozitie in terminal. Dupa acest nivel, poti folosi Claude Code eficient in orice proiect.

### Nivel Intermediar
**Durata:** ~4.5 ore | **Module:** 05, 06, 07, 08

Inveti sa automatizezi si sa extinzi: creezi skills reutilizabile, configurezi hooks care protejeaza proiectul automat, conectezi Claude la servicii externe prin MCP si folosesti sub-agenti pentru taskuri complexe. Dupa acest nivel, Claude Code devine hub-ul central al workflow-ului tau.

### Nivel Avansat
**Durata:** ~4 ore | **Module:** 09, 10

Inveti functionalitati avansate (planning mode, extended thinking, background tasks, remote agents) si cum sa impachetezi tot ce ai construit intr-un plugin distribuibil pe care altii il pot instala.
