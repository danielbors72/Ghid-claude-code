# Ghid Claude Code — Reguli de proiect

## Ce este acest proiect
Un ghid complet Claude Code in limba romana, organizat in 10 module progresive. Publicat pe GitHub si Notion.

## Limba
- Scrie tot in romana
- NU traduce: termeni tehnici, nume de concepte, comenzi, cuvinte consacrate in engleza (hooks, skills, slash commands, checkpoints, plugins, MCP, sub-agents, CLI, etc.)
- Traduce doar explicatiile si contextul narativ

## Ton
- Practic-tutorial — ca un expert pasionat care explica unui coleg inteligent
- Perspectiva din interior — scrie ca cineva care foloseste Claude Code zilnic
- "De ce te-ar interesa?" — conecteaza fiecare concept la impactul real
- Descrieri ample (4-5 randuri per concept), nu bullet points telegrafice
- Exemple concrete, nu abstractii

## Structura
- Fiecare modul e un director `NN-nume/README.md`
- Modulele au navigare prev/next si link catre cuprins
- Documente hub: README.md, HARTA-INVATARE.md, REFERINTA-RAPIDA.md, CATALOG.md
- Cross-link-uri relative (functioneaza pe GitHub)

## Reguli tehnice
- Fisierele nu se modifica fara motiv — pastreaza ce e valid
- Nu crea fisiere inutile
- Commit-uri descriptive in romana
- Nu commita .env sau settings.local.json
